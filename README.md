# 🌐 Internship Portal Frontend

This is the **Frontend** of the Internship Portal project, built using **HTML, CSS, and JavaScript**.  
It connects to the backend API to display internships, allow users to apply, and view application lists.


## 🛠 Technologies Used

- **HTML5** – Page structure and layout  
- **CSS3** – Styling and responsive design  
- **JavaScript (ES6)** – Dynamic functionality and form handling  
- **AJAX (Fetch API)** – Communication with backend REST APIs  
- **JSON** – Data format used for sending and receiving data

## 🖼 Features

- 🏠 **Home Page**
  - Introduction to available internships  
  - Navigation links to Apply and Applications pages

- 📝 **Apply Page**
  - Form for submitting internship applications  
  - Fields: Name, Email, Internship ID  
  - Connected with backend API for real-time submission

- 📄 **Applications Page**
  - Display list of all submitted applications  
  - Fetches data from backend API  
  - Dynamically updates after new submissions

 ## ▶️ How to Run
1. Clone the repository
git clone https://github.com/abinaya098/internship-portal-frontend.git⁠

2. Open the project folder in VS Code.

3. Start the backend server.

4. Open `index.html` in your browser or run using Live Server.

5. Apply for internships and view submitted applications.
